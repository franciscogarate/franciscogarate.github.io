Instalar Python3 en macOS

La forma más sencilla y recomendable de instalar la última versión de Python en macOS es utilizar Homebrew, el gestor de paquetes más popular en Mac. Así evitamos depender de la versión de Python que trae el sistema y podremos actualizarla fácilmente en el futuro.

Instalar Homebrew

Si todavía no tenemos Homebrew instalado, basta con ejecutar en la Terminal:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Al terminar, el propio instalador nos indicará un par de comandos que debemos ejecutar para añadir Homebrew al PATH. Una vez hecho, podemos comprobar que está correctamente instalado con:

brew --version

Instalar Python

Con Homebrew ya funcionando, instalar Python es tan sencillo como:

brew install python

Comprobar la instalación

Para verificar que todo ha ido bien, comprobamos la versión instalada y que el comando apunta a la instalación de Homebrew:

python3 --version

Crear un entorno virtual

Es muy recomendable trabajar siempre con entornos virtuales para aislar las dependencias de cada proyecto. Para crear uno y activarlo:

python3 -m venv .venv
source .venv/bin/activate

Con el entorno virtual activado, el comando python apuntará automáticamente a la versión del entorno, sin necesidad de crear alias.

Actualizar pip

Por último, conviene actualizar pip a su última versión dentro del entorno:

python -m pip install --upgrade pip

Y con esto ya tenemos Python listo para trabajar en macOS.
